# Network Security Lab Project

## ⚠️ Disclaimer  
This project is strictly for educational purposes and must only be used in an authorized lab environment. Do NOT use on real-world networks without explicit permission.

---

## 📌 Overview  
This project demonstrates:  
- Deployment of a controlled security testing environment  
- Network traffic monitoring  
- Packet analysis using `tshark`

The goal is to understand network behavior and security monitoring techniques.

---

## 🛠 Requirements  
- Linux (Ubuntu/Kali recommended)  
- Python 3.x (if applicable)  
- tshark (Wireshark CLI)

```bash  
sudo apt update  
sudo apt install tshark
